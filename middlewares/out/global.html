RedisSession(config)

RedisSession middleware -- created for practice
Parameters:
Name Type Description
config object the configuration object (connection, maxAge, sessionId)
Properties
Name Type Description
connection object the connection object for Redis server
Properties
Name Type Description
host string the redis host
port number the redis port
maxAge number the time to live
sessionId string the key for the session's id in the cookie

Example
// initialize the middleware
app.use(mySession({
  connection: {host: '127.0.0.1', port: 6379}, // Redis连接信息
  maxAge: 3600, // session的有效期
  sessionId: 'my.sid', // session ID 的cookie名称
}));

// set/get session variables in req.session
